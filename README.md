# FORCAP

FORCAP is a maritime enterprise vessel fleet management suite.

The product is being developed to replace legacy PROP-style ship/shore workflows with a modern, access-controlled, auditable platform.

## Initial Product Direction

- FORCAP Core
- Engineering / PMS
- Stores / Spares
- Finance / Requisitions
- Operations
- Compliance
- Knowledge / Oracle

## First Build Target

The first serious build target is FORCAP Core plus Engineering / PMS.

## Documentation

Controlled product, requirements, architecture, PMS/Class evidence, testing, security, quality, operations, and audit documentation is stored in docs/.

## Repository Status

This repository is the clean commercial product foundation. Existing FORCAP-related repos are treated as prototype/reference repositories.
