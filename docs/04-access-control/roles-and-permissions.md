# FORCAP Roles and Permissions

## Access Model

FORCAP permissions are based on:

- user identity
- role
- vessel assignment
- module access
- action-level permission
- ship/shore context

## Initial Roles

- System Admin
- Company Admin
- Fleet Manager
- Technical Manager
- Marine Superintendent
- Technical Superintendent
- Master
- Chief Engineer
- Chief Officer
- Second Engineer
- Third Engineer
- Fourth Engineer
- Electrical Officer
- Finance User
- Procurement User
- Stores User
- Auditor / Surveyor
- Supplier / Agent
- Read-only User
